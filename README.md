# Ruby CD Organizer

By Lauryn Davis + Riley Starnes 

A Ruby/Sinatra app that converts a string using some of the rules of Leetspeak.

####Technologies Used

* Ruby
* Sinatra
* HTML
* CSS

#### Setup

* Clone this repository
* Navigate to the project directory in the command line and run:
`bundle install`
`ruby app.rb`
